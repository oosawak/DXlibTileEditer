・本プログラムは百春さんのプロジェクトソースを一部改造してます。利用条件については百春さん規約が適用されます。
・640 x 480 pixelのゲームを想定してます。
・IMAGE/BG.png を背景として読み込みます。
・キー入力によりｃｓｖ、ｐｎｇの保存、ｃｓｖの読み込みができます。
　Ctl + s　ＣＳＶ保存（map.csv)
　Ctl + l　ＣＳＶ読み込み（map.csv)
　Ctl + b　ＰＮＧ保存（IMAGE/BG.png)
・コンパイルには DXLib が必要です。

　以上（oosawak)
